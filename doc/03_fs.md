# jdb_fs - 文件系统模块

定位: compio 异步文件系统封装，Direct I/O 支持。

## 依赖

```toml
jdb_alloc = { path = "../jdb_alloc" }
compio = { version = "0.17", features = ["runtime"] }
```

## 模块结构

```
jdb_fs/src/
├── lib.rs     # 模块导出
└── file.rs    # 异步文件操作
```

## 核心类型

### File

```rust
/// 异步文件封装 Async file wrapper
pub struct File {
  inner: compio::fs::File,
}

impl File {
  // 打开方式 Open modes
  pub async fn open(path: impl AsRef<Path>) -> io::Result<Self>;
  pub async fn create(path: impl AsRef<Path>) -> io::Result<Self>;
  pub async fn open_rw(path: impl AsRef<Path>) -> io::Result<Self>;

  // 读操作 Read operations
  pub async fn read_at(&self, offset: u64, len: usize) -> io::Result<AlignedBuf>;
  pub async fn read_page(&self, page_no: u32) -> io::Result<AlignedBuf>;
  pub async fn read_pages(&self, page_no: u32, count: u32) -> io::Result<AlignedBuf>;

  // 写操作 Write operations
  pub async fn write_at(&self, offset: u64, buf: AlignedBuf) -> io::Result<AlignedBuf>;
  pub async fn write_page(&self, page_no: u32, buf: AlignedBuf) -> io::Result<AlignedBuf>;

  // 同步 Sync
  pub async fn sync(&self) -> io::Result<()>;
  pub async fn size(&self) -> io::Result<u64>;
  pub async fn truncate(&self, size: u64) -> io::Result<()>;
}
```

## 设计要点

1. **基于 compio**: 全异步 IO，利用 io_uring/IOCP
2. **AlignedBuf**: 确保 Direct I/O 兼容
3. **页级接口**: 简化上层使用
4. **所有权模型**: 写操作返回 buffer 所有权 (compio 模型)
5. **零拷贝**: 避免内核态与用户态之间的数据复制
